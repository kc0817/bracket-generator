How should my spreadsheet be organized?

Put every team name in column A, starting at the top-left cell (A1).

  • One team per row — no blank rows between names.
  • Column A only; other columns are ignored.
  • Include a header like “Team” in A1 if you want — we detect and skip it automatically.

From Google Sheets: File → Download → Microsoft Excel (.xlsx) or Comma-separated values (.csv), then upload here.
